Different Ways to Write the Same Color

Same color using RGB:

rgb(0, 128, 255)

Same color using HEX:

#0080ff

Same color using HSL:

hsl(210, 100%, 50%)

Transparent Colors

Using RGBA:

rgba(0, 128, 255, 0.5)

Using HSLA:

hsla(210, 100%, 50%, 0.5)

Colors can be written using RGB, HEX, HSL, RGBA, and HSLA.
